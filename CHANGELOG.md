# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2025-12-28

### Fixed
- Fixed theme toggle icon issue where a music note was displayed instead of the moon icon (Unicode mismatch in Bootstrap Icons subset).

## [1.0.0] - 2025-12-25

### Added
- Initial public release of OmniDoc.
- Smart conversion for Sequence diagrams, network topology, and tables.
- Professional "Glassmorphism" UI.
- Standalone build system (`make.ps1`).

### Changed
- Rebranded from "DocPresent" to "OmniDoc".
- Open sourced under MIT License.
